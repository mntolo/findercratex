Finder is a multipurpose Bootstrap 5 Directory & Listings template that covers following business niches:

  1. Real Estate (property listings, filters, map, landlord / renter accounts)
  2. Car Finder (car dealer, vehicle marketplace)
  3. Job Board (for job seekers and employers)
  4. City Guide (search, map, points of interest, filters, accounts)

Each demo features a complete set of pages to allow you to build a fully functional web application in chosen category. See the list of the pages below.
